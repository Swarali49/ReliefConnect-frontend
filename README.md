# ReliefConnect – Emergency Donation Platform (Frontend)

ReliefConnect is a React-based web platform that connects donors, recipients, admins, and logistics coordinators to manage donations of essential items during emergencies. It helps organize drives so food, clothes, and medicines reach people in need on time.

## Features

- Multi-language landing page (English + Indian languages)
- Role-based dashboards: Admin, Donor, Recipient, Logistics Coordinator
- Active emergency drives with highlighted status
- Status badges for donations and requests (Pending Pickup, Picked Up, Delivered)
- Recipient feedback and star rating component
- Toast notifications for key actions (login, requests, feedback)

## Tech Stack

- React JS (Vite)
- React Router
- CSS
- React-Toastify
- Deployed on Vercel
